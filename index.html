Cambodia Air Quality Data Pipeline

Problem

Cambodia lacks centralized, cleaned air quality data infrastructure. Raw API data is noisy, unstructured, and not ready for analysis or reporting without significant processing.

Data Source

Open-Meteo Air Quality API — real-time pollutant concentrations for Phnom Penh, Siem Reap, Battambang, Kampot, and Sihanoukville.

Workflow

  1. Collect hourly air quality data via Python requests
  2. Clean and transform raw JSON using PySpark DataFrames
  3. Engineer features: AQI category, date/hour/month extraction
  4. Load into SQL Server tables (city summaries, pollutant averages)
  5. Connect Power BI for interactive dashboard reporting

Tools

PythonPySparkApache SparkSQL ServerPower BIOpen-Meteo API

Key Results

  • Automated ETL pipeline from API ingestion to SQL Server
  • Multi-city, multi-pollutant data with AQI categorization
  • Power BI dashboard for city-level air quality monitoring

What I Learned

End-to-end data pipeline development, real-time API integration, distributed processing with PySpark, and connecting analytical databases to BI tools.

ABA Transaction Analysis Dashboard

Problem

ABA Bank Telegram payment notifications are unstructured text. Extracting insights requires automated parsing and analysis.

Data Source

Exported ABA Bank payment notification messages from Telegram — transaction amounts, payment methods, customer names, and timestamps.

Workflow

  1. Export raw Telegram payment notification messages
  2. Parse text into structured data with Python regex and pandas
  3. Clean and validate extracted fields
  4. Perform exploratory analysis on revenue and customer behavior
  5. Build Power BI dashboard for transaction analytics

Tools

PythonpandasRegexJupyter NotebookPower BI

Key Results

  • Parsed 5,509 transactions totaling USD 79,347.90
  • 3,678 unique customers across 48 payment methods
  • Dashboard showing revenue performance and temporal patterns

What I Learned

Text parsing with regex, data cleaning with pandas, and translating unstructured data into actionable BI dashboards.
